## QUICK START

    - Check  
           
        java -version (11.0.9+)
           
        mvn -version (3.6.0+)

        docker -version (20.10.7+)
           
    - Build

        mvn clean package
       
    - Start app
     
       docker-compose up --build -d

    - Stop app
     
       docker-compose down
